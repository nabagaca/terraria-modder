# Randomizer

A modular randomizer for Terraria with a seed system for reproducible runs. Enable only the modules you want — each category of chaos is independently toggleable.

Press **Numpad /** to open the config panel.

## Modules

| Module | Description |
|--------|-------------|
| Chest Loot Shuffle | Items found in chests are shuffled between chest types |
| Enemy Drop Shuffle | Enemy drops are swapped between enemy types |
| Recipe Shuffle | Crafting recipes are shuffled |
| Shop Shuffle | NPC shop inventories are shuffled between vendors |
| Fishing Shuffle | Fishing catches are shuffled across fishing pools |
| Tile Drop Shuffle | What tiles drop when mined is shuffled |
| Spawn Shuffle | Enemies spawn as random types (bosses/town NPCs excluded) |
| Item Stat Scramble | Item damage, defense, and stats are scrambled |
| Starting Inventory | Starting items are randomized |
| Gravity Chaos ⚠️ | Gravity shifts between 0.25x–2.5x every 30-60 seconds |
| Weather Chaos | Weather changes unpredictably |

## Seed System

- Set a **seed** for a fully reproducible randomized playthrough
- **Seed 0**: Picks a random seed each session
- World-gen modules lock their seed when you first enter a world, keeping shuffles consistent for that world

## Keybind

| Key | Action |
|-----|--------|
| Numpad / | Toggle Randomizer config panel (in-world and title screen) |

Rebindable via Mod Menu (F6).

## Configuration

Open the Mod Menu (F6) or press Numpad / to toggle individual modules and set your seed.

## Multiplayer

Singleplayer only.

## Installation

Requires [TerrariaModder Core](https://www.nexusmods.com/terraria/mods/135).

1. Extract the zip into your Terraria folder
2. The mod goes into `TerrariaModder/mods/randomizer/`
3. Launch with `TerrariaInjector.exe`
