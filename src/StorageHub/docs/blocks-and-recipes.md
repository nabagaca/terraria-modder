# Blocks and Recipes

This page lists dedicated Storage Hub blocks/items, what they do, and current recipe coverage.

## Dedicated Blocks

| Block | Size | Purpose | Right-click Behavior |
|---|---:|---|---|
| Storage Core | 2x2 | Network root/controller | Opens Storage Hub UI (if connected network is valid) |
| Storage Drive | 2x2 | Disk container (up to 8 disks) | Shows drive interaction message |
| Storage Access | 2x2 | Access terminal | Opens Storage Hub UI |
| Storage Crafting Interface | 2x2 | Craft-focused terminal | Opens Storage Hub UI on crafting tab |
| Storage Component | 2x2 | Structural network node | No special right-click action |
| Storage Connector | 1x1 | Link piece for network routing | No special right-click action |
| Disk Upgrader | 2x2 | GUI-based disk tier upgrades | Opens disk upgrader UI |

## Item/Block Recipes (Currently Registered)

All recipes below use station: `Anvils`.

### Storage Core

- `1 Diamond + 4 Iron Bar + 10 Wood`
- `1 Diamond + 4 Lead Bar + 10 Wood`

### Storage Drive

- `5 Gold Bar + 5 Iron Bar`
- `5 Gold Bar + 5 Lead Bar`

### Basic Storage Disk

- `1 Diamond + 1 Ruby + 1 Gold Bar`

### Disk Upgrader

- `5 Gold Bar + 10 Iron Bar`
- `5 Gold Bar + 10 Lead Bar`
