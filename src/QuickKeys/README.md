# Quick Keys

Hotkeys for auto-torch, recall, quick stack, ruler, and extended hotbar.

## Features

- **Auto Torch**: Places a torch near your cursor, automatically finding the nearest valid spot
- **Auto Recall**: Instantly uses your best recall item (priority: Cell Phone > Shellphones > Mirrors > Recall Potion)
- **Quick Stack**: Quick stacks items to all nearby chests
- **Ruler**: Toggle distance overlay to measure tile distances
- **Extended Hotbar**: Use NumPad keys to quick-use items in inventory slots 11-20 (disabled by default, enable in config)

## Keybinds (Rebindable via F6 Menu)

| Default Key | Action |
|-------------|--------|
| Tilde (~) / OemTilde | Auto Torch |
| Home | Auto Recall |
| End | Quick Stack |
| K | Toggle ruler distance overlay |
| NumPad1-9 | Quick-use inventory slot 11-19 (requires Extended Hotbar) |
| NumPad0 | Quick-use inventory slot 20 (requires Extended Hotbar) |

All keybinds are rebindable via the F6 mod menu and persist across restarts.

## Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| Enabled | true | Master toggle for all QuickKeys features |
| Show Messages | false | Show chat notifications when using features |
| Extended Hotbar (Slots 11-20) | false | Enable NumPad keybinds for inventory slots 11-20 |

All settings configurable via F6 menu in-game.

## Multiplayer

Works in multiplayer.

## Installation

Requires TerrariaModder Core.

Extract this zip into your Terraria folder. The mod goes into
`TerrariaModder/mods/quick-keys/`.
